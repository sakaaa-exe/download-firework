Tutorial

Panduan lengkap untuk memulai dengan file kodingan

Langkah 1: Download File

Klik tombol Download di halaman utama untuk mengunduh file ZIP yang berisi file kodingan.

Langkah 2: Ekstrak File

Gunakan aplikasi seperti ZArchiver untuk mengekstrak file ZIP.

Setelah mendownload file ZIP nya, masuk ke ZArchiver nya dan cari file ZIPnya.

Cara mencarinya ialah masuk ke folder Download.

Setelah masuk ke folder Download, cari file ZIP nya.

Setelah itu klik file ZIP nya, setelah file ZIP nya di klik, pencet "Ekstrak ke ./Archive Name/".

Setelah itu, file ZIP sudah di ekstrak. Selanjutnya kamu membutuhkan aplikasi untuk menjalankannya, kamu bisa menggunakan aplikasi seperti Code Studioatau aplikasi koding lainnya.

Langkah 3: Menjalankan Kodenya

Setelah itu, kita harus menjalankan kodenya, kita buka aplikasi Code Studio nya di handphone kita, pada awal-awal aplikasi Code Studio akan meminta izin untuk mengakses file penyimpanan, izinkan semua permintaan aplikasinya.

Setelah itu, kita pencet menu garis tiga disebelah pojok kiri atas pada aplikasi Code Studio.

Setelah itu, pencet menu open

Cari folder download dan setelah itu klik folder download nya

Cari folder sadtyy_bot-kembang-api dan klik foldernya

Pencet file kembang-api.index

Dan setelah itu, pencet open pada file kembang-api.index

Dan setelah itu, jalankan website nya

Selamat, kamu sudah berhasil menjalankan websitenya!
